# Чпуля

Скидываешь мысль — получаешь разбор от двух ведущих менеджерского подкаста Чпуля, Миши и Лёвы. Они критикуют, развивают, спорят между собой — как в выпуске, только про твою ситуацию.

---

## Быстрый старт — просто дай ссылку своей LLM

Скопируй этот текст и отправь в любой чат с AI (ChatGPT, Claude, Gemini, и т.д.) **первым сообщением**:

```
Используй этот системный промпт для всего нашего диалога:
https://raw.githubusercontent.com/anatoly-ryzhkov/chpoolya/main/prompt.md

После того как прочитаешь — подтверди и жди мою мысль.
```

Дальше просто пиши свои мысли — AI ответит диалогом Миши и Лёвы.

---

## Установка в Claude Code (постоянный скилл `/чпуля`)

```bash
# клонируй репо
git clone https://github.com/anatoly-ryzhkov/chpoolya.git

# скопируй файлы
cp chpoolya/claude-skill.md ~/.claude/skills/чпуля.md
cp -r chpoolya/hosts ~/.claude/skills/podcast-hosts
```

После этого в любой сессии Claude Code: `/чпуля твоя мысль`

## Установка в другие инструменты

| Инструмент | Что сделать |
|---|---|
| **ChatGPT Custom GPT** | System instructions → вставить содержимое `prompt.md` |
| **Gemini CLI** | `gemini --system-prompt prompt.md` или вставить в начало чата |
| **Cursor** | Создать `.cursor/rules/чпуля.md`, вставить содержимое `prompt.md` |
| **VS Code Copilot** | `.github/copilot-instructions.md` → вставить содержимое `prompt.md` |
| **Любой другой AI** | Вставить содержимое `prompt.md` как первое сообщение |

## Структура репо

```
prompt.md            — универсальный системный промпт
claude-skill.md      — обёртка для Claude Code
hosts/
  misha_profile.md   — детальный профиль Миши
  leva_profile.md    — детальный профиль Лёвы
  topics_db.md       — тематическая база знаний из выпусков
```

База знаний пополняется с каждым выпуском подкаста.

**Ты:** у нас в команде все решения идут через меня, я уже не успеваю
